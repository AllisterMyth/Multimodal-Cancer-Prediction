# Multimodal-Cancer-Prediction
A multimodal system that takes longitudinal data in the form of CT scans, Diagnoses, Liquid biopsy to predict the risk of cancer in the next few years. The models used are CNN backbone with SWin Tiny and  ClinicalBert. Gated Multimodal fusion is used. This is trained on a synthetic dataset, will perform better on good real datasets.
The idea is inspired from Sybil by MIT. The idea is to make a model that takes a more complete form of data over a period of time to flag any problematic patterns. CNN-Swin tiny hybrid is used due to the small size of dataset I had access to but the model can be changed if needed. Gated method is used for fusion for good weighing of model expertise per decision per patient. This model is meant to work queitly in a private background on all possible data in a hospital and prompt when it finds anything. 
Due to restrictions on compute, the model is made for a single CT layer and only 2 Epochs of training but that can be changed by the user. Also to reduce compute during actual use, BERT embeddings are used rather than loading the model every time. I made the model on Kaggle but you may run it on any server that supports PyTorch
